# Decodelabs EDA Sales Analysis

## Project Overview

This project explores retail sales data to identify trends, customer purchasing behaviour, product performance, and revenue opportunities using SQL and Exploratory Data Analysis (EDA).

## Business Problem

Retail businesses need to understand which products generate the most revenue, which payment methods customers prefer, and which products drive the highest order volumes.

This analysis was conducted to support data-driven business decisions.

## Dataset

The dataset contains sales transactions including:

- Order ID
- Product
- Quantity
- Unit Price
- Total Price
- Payment Method

## Business Questions

### 1. Which products generate the highest revenue?

### 2. Which payment methods generate the most revenue?

### 3. Which products receive the most orders?

### 4. What is the average revenue per order?

### 5. Which payment method has the highest average order value?
